### 교집합, 합집합, 차집합 구하기


```python
s1 = set([1,2,3,4,5,6])
s2 = set([4,5,6,7,8,9])
```

#### 교집합


```python
s1 & s2
```




    {4, 5, 6}




```python
s1.intersection(s2)
```




    {4, 5, 6}



#### 합집합


```python
s1 | s2
```




    {1, 2, 3, 4, 5, 6, 7, 8, 9}




```python
s1.union(s2)
```




    {1, 2, 3, 4, 5, 6, 7, 8, 9}



#### 차집합


```python
s1 - s2
```




    {1, 2, 3}




```python
s2 - s1
```




    {7, 8, 9}




```python
s1.difference(s2)
```




    {1, 2, 3}




```python
s2.difference(s1)
```




    {7, 8, 9}
